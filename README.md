# PDF Extractor Railway

Extração inteligente de texto PDF com PyMuPDF e EasyOCR.

## Funcionalidades

- ✅ Detecção automática: texto nativo vs escaneado
- ✅ PyMuPDF para extração direta (textos nativos)
- ✅ EasyOCR para documentos escaneados
- ✅ Pós-processamento inteligente
- ✅ API FastAPI com documentação automática
- ✅ Deploy otimizado para Railway

## Estrutura

- `backend/` - API Python com FastAPI
- `frontend/` - Interface web
- `tests/` - Testes automatizados
